# Tokyo EYES on Milano 2025
ミラノの街に東京の地名を当てはめてみたメンタルマップです。デザインデータは Keynote ファイルになります。大阪版は絶賛作成中。

## 0. Download
 * 最新版は [Releases](https://github.com/furuhashilab/TokyoEYESonMilano2025/releases) からダウンロードできます。
